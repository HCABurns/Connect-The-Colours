The Goal

Link the Colours is a logical puzzle that involves connecting pairs of matching colours in a closed grid. The objective is to connect ALL pairs of identical colours with non-overlapping paths, using EVERY tile.

  Rules

  • Paths must consist of one or more horizontal and vertical segments to connect adjacent tiles.
  • Each colour must be connected to the other instance of the same colour using a continuous path.
  • A single tile can only be used in ONE path and any two paths CANNOT overlap or cross each other.
  • Paths must remain entirely within the grid boundaries.
  • Every tile must be used in a path, so that there are no empty tiles remaining.
Victory Conditions
  • All colours are connected to their respective identical colour, in a continuous path and without overlaps or crossovers.
Loss Conditions
  • Attempted to reassign a starting tile to a different colour.
  • Attempted to connect a tile with more than two connections.
  • Attempted to connect two tiles that are not in a straight path vertically or horizontally.
  • Attempted to connect a tile outside the provided puzzle dimensions.
  • Attempted to reassign a section of a path.
  • Attempted to connect more than one path to a starting tile.

  Game Input

Input

Line 1: Space separated integers h w representing the height and width of the puzzle.

Next h Lines: String of length w representing a portion of the puzzle.

Integers are used to define colours. Matching integers represent the same colour.

The full stop character (.) indicates an empty tile that can be used as part of a path.

Output
A single line containing integers in the form y1 x1 y2 x2 colour.
  • y1 - Y coordinate of the first tile.
  • x1 - X coordinate of the first tile.
  • y2 - Y coordinate of the second tile.
  • x1 - X coordinate of the second tile.
  • colour - Colour identifier.

Constraints
3 ≤ h, w ≤ 10
Allotted response time for first output is 3 seconds.
Allotted response time for turn outputs is 50 ms.
Turn limit is 50 turns.
